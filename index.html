Pokedex

Senne Van Reusel 2 APPAI 01

A pokedex app where you can search for Pokemon and see there stats and shiny forms. Get some info on which type is effective to which. You will see a list with all Pokemon and you can filter on name or generation.
You will be able to see the picture in real life using your camera, this will be a very basic AR. You can favorite Pokemon and this will be stored in local storage. I will use the pokeapi an api that has all the Pokemon information.
